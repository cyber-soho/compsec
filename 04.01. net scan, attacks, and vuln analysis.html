Scan Types

Ping Sweep (ICMP)
"Who is alive?" - Sends Echo Request. Timeout = Down/Filtered.
SYN Scan (Half-Open)
"Stealthy" - Sends SYN, gets SYN-ACK, never sends final ACK. Logs fewer entries.
Version Detection
"What version?" - e.g., identifying "Apache 2.4.49" to find specific CVEs.

Cyber Kill Chain (7 Stages)

1. Reconnaissance

Target research, scanning, email harvesting.

2. Weaponisation

Creating the exploit/payload (e.g., PDF with malware).

3. Delivery

Transmission via Email (Phishing), USB, Web.

4. Exploitation

Triggering the vulnerability in the system.

5. Installation

Establishing a backdoor or persistence.

6. C2 (Command & Control)

Remote control established by attacker.

7. Actions on Objectives

Data exfiltration, encryption (Ransomware), destruction.

Goal: Break the chain at any stage to stop the attack.

Malware & Social Engineering

Malware Types

Worm Self-replicating. Spreads automatically (e.g., WannaCry).
Trojan Disguised as legit software. Opens backdoors.
Ransomware Encrypts files for payment. High availability impact.

Social Engineering (Human Hacking)

The Phishing Formula

Urgency + Authority + Spoofed Branding = Click

  • Phishing: Generic fake emails.
  • Spear Phishing: Targeted using personal info.
  • Vishing: Voice/Phone based.
  • Smishing: SMS based.
